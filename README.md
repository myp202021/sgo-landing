# SGO Landing

Landing page profesional para SGO.cl — Libro de Obras Digital Chile.

**Live:** https://sgo-landing-pi.vercel.app

## Stack
- Next.js + TypeScript + Tailwind CSS
- SEO: JSON-LD schemas (SoftwareApplication, FAQPage, Organization)
- Brand: navy #1a1f3d + steel blue #3b82f6
- Deploy: Vercel

## Secciones
- Hero con foto construcción + counters
- Pain Points industria construcción
- 4 Productos (LOD, LCE, Consultoría, Gestor Documental)
- Flujo animado pipeline 6 pasos
- 3 Perfiles usuario (Constructoras, Consultoras, IFO)
- Social Proof (+700 obras, 100% MOP)
- 5 Testimonios reales
- Blog (3 artículos SEO)
- FAQ 6 preguntas
- Schemas IA
